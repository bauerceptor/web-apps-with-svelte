# Lecture 12 — Props

**Goal:** Pass data into child components with Svelte 5 props.

**Concepts covered:**
- `$props` for receiving values in a child component.
- Default prop values.
- Spreading props from an object.
- Reactive props: updating parent state updates the child automatically.

**Projects:**
- **Card Gallery:** A reusable `Card` component with title, description, and optional highlight state.
- **Profile Builder:** Edit a user object and watch the `ProfileCard` update in real time.

**Try changing:** Add a `tags` prop to the card and render them as small badges.
