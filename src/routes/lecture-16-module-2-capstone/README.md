# Lecture 16 — Module 2 Capstone: Task Dashboard

**Goal:** Combine every Svelte 5 concept from Module 2 into one app.

**Concepts covered:**
- `$state` and `$derived` for tasks and stats.
- Props and callback events for a reusable `TaskCard` component.
- `{#if}` for empty states and filters.
- Keyed `{#each}` for the task list.
- `bind:value` for adding and editing tasks.

**What it does:**
- Adds, toggles, edits, and deletes tasks.
- Filters tasks by all, active, or completed.
- Shows live counts and derived stats.

**Try changing:** Add a search box that filters tasks by title.
