# Lecture 14 — Events

**Goal:** Respond to user actions with DOM events and component callbacks.

**Concepts covered:**
- DOM events such as `click`, `submit`, and `keydown`.
- Event methods like `preventDefault` and `stopPropagation`.
- Callback props for child-to-parent communication in Svelte 5.

**Projects:**
- **Event Playground:** A form and nested boxes that show how event handling and propagation work.
- **Custom Stepper:** A reusable stepper component that sends changes back to the parent through props.

**Try changing:** Add a keyboard shortcut that resets the stepper when the Escape key is pressed.
