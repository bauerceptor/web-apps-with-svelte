# Lecture 13 — Logic Blocks

**Goal:** Control what renders with Svelte logic blocks.

**Concepts covered:**
- `{#if}` for conditional rendering.
- `{#each}` keyed loops for lists with stateful items.
- `{#await}` for async data and its loading, success, and error states.

**Projects:**
- **Async Quote:** Fetch a quote from a public API with a static fallback if the network fails.
- **Keyed Todo List:** Reorder todos and see why a stable key keeps each item's state intact.

**Try changing:** Add a filter so only incomplete todos are shown.
