# Lecture 15 — Bindings

**Goal:** Keep DOM elements and component state in sync with two-way bindings.

**Concepts covered:**
- `bind:value` for text inputs, textareas, and selects.
- `bind:checked` for checkboxes.
- `bind:group` for radio buttons and checkbox groups.
- `bind:this` for element references.

**Projects:**
- **Settings Form:** Bind every control to a single settings object and watch it update live.
- **Survey Poll:** Use `bind:group` to collect votes and display a live bar chart.

**Try changing:** Add a theme toggle and bind it to a CSS class on the page body.
